# CoopCredit Frontend

## Overview
The `coopcredit-frontend` is the web interface for the CoopCredit system. It allows Affiliates to register, login, and manage their credit applications. Analysts can view and evaluate applications.

## Tech Stack
- **Framework**: React 18
- **Build Tool**: Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **State Management**: React Query (TanStack Query)
- **Routing**: React Router DOM 6
- **Forms**: React Hook Form + Zod
- **Icons**: Lucide React

## Project Structure
- `src/api`: Axios client and API modules.
- `src/components`: Shared UI components (Input, Button, etc.).
- `src/contexts`: Global state (AuthProvider).
- `src/pages`: Application views (Login, Register, Dashboard).
- `src/routes`: Route definitions and Protection wrappers.
